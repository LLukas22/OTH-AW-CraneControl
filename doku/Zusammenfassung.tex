\chapter{Zusammenfassung}


\section{Stand bei Abgabe}
Alle geforderten Gesten konnten implementiert und zuverlässig detektiert werden. Das Programm befindet sich in einem Zustand, indem es nicht weiter entwickelt werden muss. Das Versenden der Daten über den TCP/IP-Server ermöglicht eine flexible Einsatzmöglichkeit.

\section{Ausblick}
Nun muss auf der Steuerung des Kranes ein TCP/IP Client implementiert werden, der die Daten ausliest. Dies sollte in einem Thread erfolgen, der unabhängig von der bereits implementierten Kransteuerung ist. Ist das System als ganzes einsatzfähig, so kann es als Vorzeigeobjekt bei Veranstaltungen, wie den Tag der offenen Tür dienen.
